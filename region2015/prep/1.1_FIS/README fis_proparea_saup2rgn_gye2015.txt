1. Institución recolectora original (cuál institución hizo la recolección de los datos de su fuente).
Code of Country http://www.codeofcountry.com/country-codes y se conservó las áreas sugeridas por la metodología global ya establecidas para cada provincia. 

2. Institución que provee los datos a este proyecto (si es distinta de la recolectora).
Asociación Consulsua-Biótica

3. Citas detalladas de la fuente, si se trata de uno o más artículos científicos.
No aplica.

4. Método de captura de los datos. Si fue un estudio realizado para obtener esos datos, si se hizo utilizando algún instrumento de medición, cuál método se aplicó, si se utilizó una encuesta o formulario para recogerlos, etc.
A partir de la confirmación del código de área, se estableció los valores sugeridos por la metodología global con respecto a la proporción de área para las provincias del Golfo de Guayaquil. 

5. Período de tiempo que cubren los datos (año desde y año hasta). O enumeración de años si no son consecutivos.
No aplica

6. Unidad en la cual vienen los valores.
Código numérico de hasta tres dígitos indicado en el See Around Us Project (SAUP) basados en el ISO 3166-1 que es la primera parte de normas estándar internacionales de la ISO 3166 publicado por la organización internacional de normalización (ISO), que proporciona códigos para los nombres de países y otras unidades administrativas. El estándar ISO 3166 primero fue publicado en 1974 por la organización internacional de normalización (ISO) y amplió a tres partes en 1997, de las cuales la primera parte corresponde al anterior partido único.

7. Significado o explicación de qué significan los valores.
A través de un código de área universal se asignan valoraciones para las distintas provincias del Ecuador.

8. Observación o comentario sobre los datos.
Ninguna

9. Significado y explicación de las columnas que se incluyen en el archivos, solo si son diferentes de las que se especifican en la aplicación OHI-ecu o si la capa de datos es nueva, creada ad hoc para este estudio regional. 
El código de área asignado para Ecuador es 218 por esta base web, el mismo fue asignado para las provincias que componen el Golfo de Guayaquil, con respecto a los valores de áreas par acada provincia se mantienen los valores sugeridos por la metodología global para Ecuador.
